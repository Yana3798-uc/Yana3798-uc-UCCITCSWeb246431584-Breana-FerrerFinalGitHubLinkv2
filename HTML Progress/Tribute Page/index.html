HTML Basics

HTML stands for HyperText Markup Language. It creates the structure of web pages.

Every HTML page starts with a DOCTYPE declaration and html tag. The head holds metadata. The body contains visible content.

Tags come in pairs like opening and closing. Some tags like img are self-closing.

<!DOCTYPE html>

Elements can have attributes. The class and id attributes help with styling.

  • Use semantic tags like header, nav, main, and footer
  • Always validate your HTML code
  • Nest elements properly
CSS Selectors

CSS selectors target HTML elements. The basic selector is the element name.

Class selectors use a dot. They target elements with that class name.

.container { background: white; }

ID selectors use the hash symbol. Each ID should be unique on a page.

  • h1 targets all h1 elements
  • .menu targets elements with class menu
  • #header targets element with id header

Attribute selectors target based on attributes like [type="text"].

Box Model

Every HTML element is a rectangular box. It has content, padding, border, and margin.

Content is the actual text or image. Padding creates space inside the border.

box-sizing: border-box;

Border sits between padding and margin. Margin creates space outside the border.

  • Width includes content + padding + border
  • Use border-box for easier sizing
  • Margin collapses between elements

Box-sizing property changes how width and height get calculated.

Flexbox Layout

Flexbox makes it easy to align items in one dimension. Set display to flex on the container.

Flex-direction controls row or column layout. Justify-content aligns on the main axis.

display: flex; justify-content: space-between;
  • Works great for navigation bars
  • Centers items easily
  • Handles different screen sizes

Align-items controls cross-axis alignment. Flex-wrap allows items to wrap.

Responsive Design

Responsive design adapts to different screen sizes. Use relative units like percentages.

Media queries apply styles at specific breakpoints. Viewport meta tag is essential.

@media (max-width: 768px) { ... }
  • Start with mobile-first approach
  • Test on real devices
  • Use flexible grid systems

Flexible images scale with containers. Max-width of 100% prevents overflow.

CSS Animations

CSS animations bring elements to life. Use keyframes to define animation steps.

Transitions create smooth changes on hover or focus. They work with most properties.

transition: all 0.3s ease;
  • Keep animations short and subtle
  • Use transform for better performance
  • Animation-fill-mode controls end states

Transform properties move, rotate, and scale elements efficiently.
